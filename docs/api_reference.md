# API Reference

*This documentation is currently under construction.*

For now, please refer to the following sources:
- **Rust SDK**: See `memsdk/src/lib.rs` docs or run `cargo doc --open`.
- **JS SDK**: See `js-sdk/README.md`.
- **CLI**: Run `memcli --help` or `memcli [command] --help`.

## Core Concepts
- **Block**: The fundamental unit of storage.
- **Node**: A single running instance of MemCloud.
- **Peer**: Another node on the network.
